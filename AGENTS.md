# AGENTS.md

AI エージェント向けの操作指示書。

## エージェントの基本原則

1. **破壊的操作は必ず確認してから実行する** — ファイル削除・DB 操作・`git reset --hard` など
2. **ソースコード中の全てのコメントは英語で書く**
3. **ソースコードのアーキテクチャは functional-core-imperative-shell に従う**
4. **ソースファイル毎に実装したら必ずテストを作成し、実行する**
5. **実装完了したら必ずドキュメントを更新する**
6. **実装完了後、ドキュメント更新後のタイミングで必ず`git`にコミットする**
